# 多语言模块

## 模块简介
引入此模块为项目适配多语言，项目中使用的字符串都需从此模块中引用。

## 使用说明
### 新增一个字符串
新增一个字符串，需要在已存在所有多语言文件`strings.xml`中进行翻译。

### 新增一种多语言
新增一种多语言，在对照表中根据语言名称查找新增语言的语言代码，并在res文件夹下新增目录，目录格式为：
```
value-语言代码-r地区代码
```
复制values目录下`strings.xml`到新增目录下，并翻译所有字符串。
